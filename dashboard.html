What's PublicMe?

  • PublicMe is a free and open tool for exposing local servers to public network
  • It can expose TCP protocols, such as HTTP, SSH, all major databases

  • How to install/update

    $

    publicme HTTP port 80

  • Authenticate

    First obtain auth token from publicme.uz/api/auth/, then

    $

    publicme.exe AUTH <add-authtoken>
